Admin sticky notes for front end page and posts

== Description ==

Front end sticky notes plugin for WordPress. Notes are only displayed to administrators.

== Installation ==

1. Upload the `ml-sticky-notes` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. That's it. There is no admin page, just start adding your notes on the front end (you need to be logged in as an admin) - see screen shot
